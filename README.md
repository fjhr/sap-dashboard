# SAP B1 Sales Dashboard

Dashboard de ventas de SAP Business One (últimos 5 días), publicado en GitHub Pages y alimentado por Google Apps Script.

## Stack

- **Backend**: Google Apps Script → consulta SAP B1 Service Layer, cachea cada 6h
- **Frontend**: HTML + Tailwind CSS + Chart.js → GitHub Pages
- **Hosting**: `https://fjhr.github.io/sap-dashboard`
